# Noctalia Unofficial Plugins

A collection of unofficial plugins for [Noctalia](https://noctalia.net/), a modern Linux desktop environment.

## Plugins

### ScreenShot & Record in Hyprland

A screenshot plugin with OCR, Google Lens integration, and screen recording. Only supports Hyprland.

- **Features**: Screenshot capture, OCR, Google Lens integration, screen recording
- **Version**: 1.0.0

### Hyprland Submap

Shows the current Hyprland Submap if it isn't default.

- **Features**: Displays active submap in the bar
- **Version**: 1.0.0
