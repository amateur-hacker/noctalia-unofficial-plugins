# Hyprland Submap in Noctalia Shell

This is an implementation that shows the current Hyprland submap you are in. I use Hyprland submaps for certain tasks such as resizing windows, temporarily disabling keybindings to avoid conflicts with other applications, and other workflows. Because of this, I needed a way to easily see which submap is currently active.

To implement this feature, I used AI (Claude) to better understand how to use IPC and establish the connection properly. This helped me build the functionality in a more efficient way.
